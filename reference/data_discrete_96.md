# Discrete 96-well plate dataset

A dataset containing 22 conditions for a hypothetical drug treatment in
a 96-well plate. The first column contains the negative control while
the last column contains the positive control. Each treatment is
assigned to four wells in a 96-well plate.

## Usage

``` r
data_discrete_96
```

## Format

A data frame with a `Value` and a `well` column.

## Source

Drugs were chosen from a standard FDA approved drug library.
