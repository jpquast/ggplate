# Discrete 6-well plate dataset

A dataset containing 3 conditions for a drug treatment in a 6-well
plate. The treatment consists of a DMSO negative control, Rapamycin and
Taxol. Each treatment is assigned to a position in a 6-well plate.

## Usage

``` r
data_discrete_6
```

## Format

A data frame with a `Value` and a `well` column.

## Source

Theoretically possible treatment for cells.
