# Continuous 1536-well plate dataset

A dataset containing 1536 positive numeric values randomly generated
using a normal distribution
([`rnorm()`](https://rdrr.io/r/stats/Normal.html)). Each value is
assigned to a position in a 1536-well plate.

## Usage

``` r
data_continuous_1536
```

## Format

A data frame with a `Value` and a `well` column.

## Source

Randomly generated.
